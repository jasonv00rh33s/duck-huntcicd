# Duck Hunt

Гра Duck Hunt реалізована на Python з використанням бібліотеки pygame.

## Команда
- Шлапа Уляна Андріївна — гілка `engine`
- Цехановський Даніель 2 — гілка `duck-logic`
- Базюк Маркіян — гілка `ui-ux`
- Камінський Владислав — гілка `config`

## Встановлення
```
pip install pygame
```

## Запуск
```
python game.py
```

## Аргументи командного рядка
```
python game.py --difficulty easy    # легкий рівень
python game.py --difficulty normal  # нормальний (за замовчуванням)
python game.py --difficulty hard    # важкий
python game.py --fps 30             # змінити FPS
```
